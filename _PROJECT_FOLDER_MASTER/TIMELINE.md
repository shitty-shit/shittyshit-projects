## Date Policy

The system preserves multiple kinds of time instead of treating file modified time as the only truth.

## Date Types

- Original date: when the thing actually began or happened.
- Captured date: when the note, recording, export, or file was created.
- Imported date: when it entered `.PROJECTS_SHITTYSHIT`.
- Processed date: when an agent classified or summarized it.
- Reviewed date: when Kevin or an agent judged its meaning/status.
- Target date: when it should be published, launched, revisited, or moved.
