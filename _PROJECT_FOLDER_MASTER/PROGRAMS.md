# Programs

Version: 0.1
Created: 2026-05-10

## Purpose

This file tracks tools, agents, hooks, and repeatable programs that may operate on `.PROJECTS_SHITTYSHIT`.

## Baseline Programs

### Folder Inventory

Reads top-level project folders and updates `PROJECT_INDEX.yaml`.

Status: proposed

### Project Card Creator

Creates a local project markdown file inside a selected project folder using the existing global template.

Status: proposed

### Date Preserver

Records observed folder dates separately from imported, processed, and reviewed dates.

Status: proposed

### Project Review Clock

Runs a periodic review pass that asks what changed, what matters, and what the next action is.

Status: proposed
