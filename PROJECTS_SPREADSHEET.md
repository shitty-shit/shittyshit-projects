## How to Add/Update

### Add New Project
1. Add new row at bottom of table
2. Fill all columns:
   - **Name:** Full project name
   - **Codename:** UPPERCASE_WITH_UNDERSCORES
   - **Status:** ACTIVE/PARKED/MUDROOM
   - **Type:** content/code/infra/creative/ai/photo
   - **Revenue:** YES/NO
   - **Priority:** 1 (highest) - 5 (lowest)
   - **Tags:** #tag1 #tag2 #tag3
   - **Last Updated:** YYYY-MM-DD
   - **Next Action:** One clear next step

### Update Status
1. Find project in table
2. Update any field as needed
3. Change Status when project moves
4. Update Last Updated date
