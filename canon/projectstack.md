## 🧬 The Harnesses Thesis

MCP connected tools to models — that was the last gold rush.
Harnesses connect models to models, agents to agents, commands to pipelines — that is the next one.

A **Harness** = lightweight command contract defining:
- Which models participate
- What syntax each receives
- How outputs reconcile
- Where reconciled output routes

No n8n. No LangChain. No Docker. Just contracts and clipboard.
